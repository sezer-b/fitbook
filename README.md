# spring-boot-angular-blueprint

A blueprint for full-stack web applications built with **Spring Boot**, **Angular**, and **PostgreSQL**. Containerized with Docker for straightforward deployment.

Use this as a starting point for new projects instead of setting up the same boilerplate from scratch every time.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Java 25, Spring Boot 4 |
| Frontend | Angular 21 |
| Database | PostgreSQL |
| Containerization | Docker, Docker Compose |

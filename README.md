# FitBook

A fitness marketplace where providers list courses and users discover and book local classes.

Browse yoga, pilates, CrossFit and more from independent trainers and studios near you, all in one place.

## Features

**For users**
- Browse fitness courses by location, category, and availability
- Book individual class slots and manage upcoming sessions
- Join a waitlist when a class is full
- Google and Apple ID login

**For providers**
- Create courses with flexible recurring schedules (e.g. every Tuesday and Thursday)
- Configure multiple time slots per day
- Set capacity per slot and manage bookings
- Cancel or add individual slots outside the regular schedule
